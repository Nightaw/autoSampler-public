# Summary / 摘要

## English

`autoSampler-public` is a compact worker-style demo for post-capture media quality review.  
It models sample-unit ingestion, device selection, scenario execution planning, heuristic label validation, storyboard evidence generation, and structured report delivery through both CLI and API entry points.

## 中文

`autoSampler-public` 是一个面向采样后处理链路的轻量公开演示项目。  
仓库覆盖 sample unit 读取、设备选择、场景执行规划、标签规则校验、storyboard 证据生成，以及通过 CLI 和 API 输出结构化报告。
